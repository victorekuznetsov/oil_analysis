## 4. Структура дашборда (6 вкладок)

### Вкладка 1 — Обзор (Overview)

**KPI-карточки:**
1. Всего проб (в фильтре)
2. % «Допустимое» / «Требует мер» / «Недопустимое»
3. Уникальных единиц техники
4. Средний TBN / средний TAN по выборке
5. Проб с критичными металлами (Fe/Cu/Si > порога)
6. Последняя дата отбора

**Графики:**
1. **Матрица здоровья парка** — heatmap (Гаражный № × параметр), цвет = статус последней пробы
2. Доля статусов — donut (Допустимое/Требует мер/Недопустимое)
3. Динамика проб по месяцам — stacked bar по статусу
4. Топ-15 «проблемных» единиц — горизонтальный bar (кол-во «Требует мер» + «Недопустимое»)
5. Распределение по узлам — bar (ДВС/ГС/КПП/МКЛ/МКП…)
6. Пробы по маркам масла — pie

### Вкладка 2 — Металлы износа (Wear Metals)

**Источник:** col 22–30 (Cu, Fe, Cr, Pb, Sn, Al, Ni, Ti, V) + Si (41).

**Графики:**
1. **Тренд Fe по единице** — line + пороговые зоны (зелёная/жёлтая/красная), маркеры замены масла
2. Тренд Cu — line + пороги
3. Мультиметалл-обзор — small multiples (Fe, Cu, Cr, Pb, Al, Si) для выбранной единицы
4. **Нормализация к наработке масла** — металл / моточас масла (скорость износа), scatter
5. Тепловая карта металлов (единица × элемент) — по последней пробе
6. Boxplot Fe по маркам техники — сравнение парков
7. Корреляция Fe ↔ Si (scatter) — абразивный износ от пыли
8. Топ-20 проб с макс. Fe/Cu — таблица с подсветкой

### Вкладка 3 — Присадки и деградация (Additives & Degradation)

**Источник:** Ca, Mo, P, Zn, Mg, Ba, B (присадки) + TBN, TAN, OXI, NIT, V100, V40, VI.

**Графики:**
1. **TBN vs TAN во времени** — dual line, точка пересечения = сигнал замены
2. Остаточный ресурс присадок — Zn/P/Ca тренд (истощение)
3. **Деградация масла** — OXI + NIT + сажа на одной оси времени
4. Вязкость V100/V40 + индекс VI — контроль сдвига вязкости
5. Скорость падения TBN (ppm/1000 м/ч) — bar по единицам, предиктор интервала замены
6. Scatter TBN vs наработка масла — линия тренда + прогноз пересечения порога

### Вкладка 4 — Загрязнения (Contamination)

**Источник:** Si (41), Na (39), K (40), вода W (42), гликоль (43), топливо (44), сажа (45), ISO 4406 (54).

**Графики:**
1. **Si (пыль/абразив)** — тренд + порог (диагностика воздушного фильтра)
2. Na + K (антифриз) — тренд, всплеск = пробой системы охлаждения
3. Гликоль — bar событий > 0 (критично: прокладка ГБЦ)
4. Вода — тренд + порог
5. Разжижение топливом (F, %) — тренд (форсунки/ЦПГ)
6. **Сажа (ST)** — тренд + порог (характерно для QSK50 под нагрузкой)
7. Класс чистоты ISO 4406 — динамика (гидравлика/редукторы)
8. Комбо-индикатор «пробой ОЖ»: Na↑ + K↑ + гликоль↑ → подсветка

### Вкладка 5 — Предиктивная аналитика (Predictive)

**Алгоритмы:**
- Линейная регрессия (`scipy.stats.linregress`) тренда параметра → прогноз пересечения порога
- Z-score аномалий (± 3σ по истории единицы)
- EWMA-сглаживание (α=0.2)
- Расчёт **Oil Health Score (0–100)** — взвешенный композит

**Health Score (черновик весов):**
```
score = w·score_param(...)
  TBN            20%   (остаточный ресурс)
  Fe             15%   (износ ЦПГ)
  Cu             10%
  Si             15%   (абразив)
  Сажа           10%
  TAN            10%
  Вода+гликоль   10%   (пробой ОЖ)
  Вязкость V100  10%
```

**Графики:**
1. Gauge Health Score по выбранной единице + история
2. **Прогноз TBN** — линия + доверительный интервал + дата достижения порога
3. Прогноз Fe/Cu — регрессия + прогноз ремонта
4. Таблица риска: единица / параметр / текущее / тренд (↑↓→) / прогноз даты / риск (Н/С/В)
5. Ранжирование парка по Health Score — bar (худшие сверху)
6. Аномалии — scatter с выделенными точками > 3σ

## 5. Ключевые диагностические паттерны (для подсказок в UI)

| Паттерн | Сигнатура в данных | Вероятная причина |
|---------|--------------------|-------------------|
| Абразивный износ | Si↑ + Fe↑ + Al↑ | пыль, негерметичность впуска/фильтра |
| Пробой прокладки ГБЦ | Na↑ + K↑ + гликоль > 0 + вода↑ | охлаждающая жидкость в масле |
| Разжижение топливом | F↑ + V100↓ + TBN↓ | форсунки, неполное сгорание |
| Износ подшипников | Cu↑ + Pb↑ + Sn↑ | вкладыши коленвала |
| Истощение масла | TBN↓ + TAN↑ + OXI↑ | пора менять масло |
| Перегрев / нагар | сажа↑ + NIT↑ | работа под нагрузкой, EGT (связка с DML) |

> 🔗 **Связка с DML:** высокая сажа/нитрование в масле коррелирует с высоким EGT цилиндров (см. отчёты по DML). Возможна кросс-панель «масло × EGT» на будущее.
