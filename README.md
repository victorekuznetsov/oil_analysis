# oil_analysis — Дашборд анализа масла (QSK50 / горная техника)

Рабочий репозиторий для дашборда спектрального анализа масла парка горной техники.
Перенесено из `victorekuznetsov/nte200` (ветка `claude/qsk50-predictive-dashboard-XPVP6`).

## Содержимое

| Файл | Описание |
|------|----------|
| `OIL_DASHBOARD_SPEC.md` | **Главный документ** — спецификация дашборда (структура столбцов, пороги, 6 вкладок, предиктив, диагностические паттерны) |
| `Свод_анализов_масла.xlsm` | Основная база проб (лист `Анализы`: 43 251 строка, 79 столбцов, header в строке 2) |
| `Свод_анализов_масла 21.05.xlsm` | Более ранняя версия базы (от 21.05) — уточнить, какая основная |
| `Доливки Горная Евразия.xlsx` | Доливки масла (для отдельной вкладки/оверлея) |
| `requirements.txt` | Зависимости Python |

## Быстрый старт

```bash
pip install -r requirements.txt
# разработка дашборда согласно OIL_DASHBOARD_SPEC.md
# streamlit run oil_dashboard.py
```

## Проверенный парсер базы

```python
import pandas as pd

df = pd.read_excel('Свод_анализов_масла.xlsm', sheet_name='Анализы',
                   header=1, engine='openpyxl')
df['Состояние'] = df['Состояние'].astype(str).str.strip().str.capitalize()
# NTE200 двигатели → 1610 проб
nte = df[(df['Марка'] == 'NTE 200') & (df['Узел'] == 'ДВС')].copy()
```
